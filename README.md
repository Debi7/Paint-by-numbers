# «Рисуем по номерам» — это наборы для рисования, которые позволяют создавать картины, раскрашивая пронумерованные участки холста красками соответствующего цвета. Это хобби, которое не требует навыков рисования, так как каждый цвет краски соответствует номеру на схеме, и в результате получается готовое произведение.

# Чтобы точно удостовериться, какой именно цвет должен быть на пронумерованных участках холста, и тем более на совсем мелких не пронумерованных участках — можно кликнуть на любой участок картины и получить в строке под картиной информацию о том, какой цвет в этом месте нужно использовать.
